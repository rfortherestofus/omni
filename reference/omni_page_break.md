# Create page break

Creates a page break page for the pdf report

## Usage

``` r
omni_page_break(page_break_title, color, nmbr_pattern = 1)
```

## Arguments

- page_break_title:

  title of the page break

- color:

  Desired color. Must be one of 'yellow', 'teal', 'orangered',
  'periwinkle', 'plum'.

- nmbr_pattern:

  Desired pattern number. Most colors (except Teal) have only one.
