# Print an Omni table in an R Markdown document

Print an Omni table in an R Markdown document

## Usage

``` r
# S3 method for class 'omni_table'
knit_print(x, ...)
```

## Arguments

- x:

  An object created by \`omni_table()\`.

- ...:

  Additional arguments passed to flextable's print method.

## Value

The rendered table output.
