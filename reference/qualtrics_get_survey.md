# Get survey by name of ID

Get survey by name of ID

## Usage

``` r
qualtrics_get_survey(survey_id_name, convert = FALSE, var_labels = FALSE, ...)
```

## Arguments

- survey_id_name:

  Survey name or ID

- convert:

  Convert characters, default to FALSE

- var_labels:

  qualtRics::fetch_survey add_var_labels parameter, default to FALSE

- ...:

  Others args

## Value

A survey
