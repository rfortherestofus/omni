# Omni HTML Report

Omni HTML Report

## Usage

``` r
html_report(
  main_font = NULL,
  background_color = NULL,
  remove_logo = FALSE,
  use_csi_style = FALSE,
  ...
)
```

## Arguments

- main_font:

  Main font for the report

- background_color:

  Background color

- remove_logo:

  Whether to remove Omni logo or not

- use_csi_style:

  Whether to use CSI style or not

- ...:

  Params to pagedown::html_paged

## Value

An rmd format
