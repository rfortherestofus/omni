# Set the page counter in Omni pdf reports

Controls the page counter for Omni PDF reports.

## Usage

``` r
omni_set_page_start_pdf(start_from, counter_start, total_pages = 100)
```

## Arguments

- start_from:

  The page on which the counter should restart

- counter_start:

  The value of the first page

- total_pages:

  Number of pages to generate CSS for (default: 100)

## Value

An HTML character for rmarkdown
