# Write API key to the system - run only once

Write API key to the system - run only once

## Usage

``` r
qualtrics_register(organization_name = "OMNI", api_key, base_url)
```

## Arguments

- organization_name:

  Name of your organization (e.g. OMNI)

- api_key:

  Your Qualtrics API key

- base_url:

  Base URL

## Value

Env variables
